# Power BI Dashboard

This folder is reserved for Power BI dashboard assets for the Enterprise Urban Mobility Data Platform.

The dashboard will connect to the PostgreSQL analytics views created by the SQL layer.

## Dashboard File

```text
enterprise_urban_mobility_dashboard.pbix
```
The .pbix file will be created later using Power BI Desktop.

## Dashboard Documentation
Dashboard design, pages, visuals, and data sources are documented in:
```text
docs/powerbi_dashboard_guide.md
```

## Screenshot Folder
After the dashboard is built, exported screenshots should be added to:
```text
dashboard/powerbi/screenshots/
```
These screenshots can be referenced from the main README so recruiters and reviewers can view dashboard results without opening Power BI Desktop.
